## Descripción
Este repositorio contiene una práctica básica de uso de Git, donde se realizaron tres commits principales para documentar el proceso de trabajo con control de versiones.

---

## Commits realizados

1. Primera versión: Comandos básicos hasta subir el repositorio a GitHub (`git push origin master`).  
2. Segunda versión: Nuevos comandos para gestionar historial, cambios y archivos.  
3. Tercera versión: Documentación completa de comandos Git organizados por categorías.

---

## Propósito
Aprender el flujo de trabajo de Git: inicialización, seguimiento, commits, manejo remoto y documentación.
